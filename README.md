# using-algs4

## Windows

Enviroment Variables

Variable                | Value    
------------------------|----------
CLASSPATH               |   C:\path\to\algs4.jar
JAVA_HOME               |   C:\path\to\jdk-1.2.3.4\
path                    |   C:\path\to\jdk-1.2.3.4\bin\

now you can import from `import edu.princeton.cs.algs4.*;`
